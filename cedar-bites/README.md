# Cedar Bites Ecommerce Website

Next.js App Router website for Cedar Bites.

## Pricing rules

- Single jar: AED 33
- Bundle Pack - 4 Jars: AED 125
- Dubai delivery: AED 15
- Free Dubai delivery from AED 125 subtotal

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
